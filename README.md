# Agentic Data Scientist for Distributed ML Pipelines

LLM-based agent that autonomously constructs ML pipelines from
Apache Spark dataset statistical profiles.

---

## Quick Start (8 commands)

```bash
# 1. Clone / enter the project
cd agentic-ml-pipeline

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set your OpenAI API key
export OPENAI_API_KEY="sk-..."

# 4. Download all four benchmark datasets
python scripts/download_datasets.py

# 5. Run all main experiments (B1, B2, B3, Agent)
python experiments/experiment_runner.py

# 6. Run ablation studies (A1, A2) — must run AFTER step 5
python experiments/ablation_runner.py

# 7. Run analysis → generates all tables for the paper
python notebooks/03_analysis.py

# 8. Check results
cat experiments/results/table3_main_results.csv
```

## Key Design Decisions

**Temperature = 0.0** — Always. Never change this for reported experiments.
The LLM must be deterministic. Verify by calling each profile 3x and
confirming identical output before starting experiments.

**B3 is the most important baseline.** If the agent does not beat B3,
the LLM adds zero value over simple rules. B3 uses the same profile
statistics as the agent, just deterministic rules instead.

**Profiles are saved to disk** after the first profiling run.
The ablation runner loads these saved profiles — it does NOT re-profile.
This ensures ablation conditions receive identical statistical inputs.

**results.csv is append-only.** Running the experiment runner twice
will add duplicate rows. If you need a clean run, delete results.csv first.

## Output Files

| File | Contents |
|------|----------|
| `results.csv` | One row per (dataset, method, fold) run |
| `agent_log.jsonl` | One JSON line per LLM call (full prompt + response) |
| `profiles/*.json` | Saved dataset profiles for reproducibility |
| `table3_main_results.csv` | Paper Table 3: main results pivot |
| `wilcoxon_tests.csv` | Wilcoxon W, p-value, Cohen's d per baseline comparison |
| `table4_agent_decisions.csv` | Agent Decision Map (model + reasoning per dataset) |
| `table5_ablation.csv` | Ablation A1 and A2 deltas |
